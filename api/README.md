# Night Club API

## Installation

```bash
npm install
```

## Running the API

```bash
npm start
```

The API will be available at http://localhost:4000.

## Documentation

Docs are located at http://localhost:4000.

**Note:** Authentication has been removed from this API. You can access all endpoints without an access token.

Read about `json-server` to learn how to do things like pagination and limitation on https://www.npmjs.com/package/json-server-relationship
